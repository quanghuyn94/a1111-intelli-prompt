# All Intelli Prompt hotkeys

| Key         | Description                                  |
|----------------|----------------------------------------------|
| `↑` and `↓`        | Navigate between suggestions                 |
| `Tab` or `click`  | Auto-complete                                |
| `Esc`            | Close suggestion content                     |
| `Shift` + `Space`   | Open suggestion content                      |
| `→`| Fill with current suggestion |
